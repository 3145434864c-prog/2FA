# Sistema de Gestión de Inventarios – NUVA S.A.S.

![Banner de inventario](https://raw.githubusercontent.com/3145434864c-prog/2FA/bc1a8e4f624392b2188a9b17dbd366acdbb42053/inventario.png)


## Descripción del sistema
El **Sistema de Gestión de Inventarios de NUVA S.A.S.** es una solución diseñada para facilitar el control y la administración de productos dentro de la empresa.  
Permite llevar un registro claro de los artículos disponibles, así como de los movimientos de entrada y salida, ayudando a mantener la información organizada, actualizada y confiable.

Este sistema está pensado como una herramienta práctica para apoyar las operaciones diarias y mejorar la visibilidad del inventario.

## Objetivo del proyecto
Desarrollar un sistema que permita **gestionar inventarios de manera eficiente**, reduciendo errores manuales y proporcionando información clara para la toma de decisiones operativas.

## Alcance
- Registro y gestión de productos
- Control de entradas y salidas de inventario
- Consulta del stock disponible
- Estructura preparada para futuras mejoras y ampliaciones

## Tecnologías utilizadas
- **Backend:** PHP 8
- **Base de datos:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## Beneficios del sistema
- Mayor control y visibilidad del inventario.
- Reducción de errores en el manejo manual de productos.
- Información centralizada y organizada.
- Apoyo a la toma de decisiones operativas.
- Base sólida para el crecimiento y mantenimiento del sistema.

## Buenas prácticas aplicadas
- Organización clara del código.
- Separación de responsabilidades.
- Enfoque en mantenibilidad y escalabilidad.
- Desarrollo orientado a necesidades reales del negocio.

## Posibles mejoras futuras
- Gestión de usuarios y autenticación.
- Roles y permisos.
- Reportes y estadísticas de inventario.
- Integración con otros sistemas.
